# 《高效能人士的七个习惯》Wikipedia 摘要

这是一份基于 Wikipedia 条目的转述性摘要（非逐字摘录）

## 概览

- 这是 Stephen R. Covey 撰写的商业/自助类书籍，1989 年首次出版。
- 条目概述其核心意图：通过“七个习惯”推动并滋养个人改变，并讨论“有效性（effectiveness）”与价值系统的选择。
- Wikipedia 条目提到：作者强调以“品格伦理（character ethic）”为中心，而不是以“人格魅力/形象（personality ethic）”为中心。

## 七个习惯

1. **Be proactive（积极主动）**
   - 要点：对自己的反应负责，主动采取行动让处境变好。
   - Wikipedia 条目提到的表述框架：刺激与反应之间存在“选择如何回应”的空间；并区分“关注圈（circle of concern）”与“影响圈（circle of influence）”，鼓励把精力放到可影响之处。

2. **Begin with the end in mind（以终为始）**
   - 要点：先在心里“看见”想要的未来，再倒推计划与行动。
   - Wikipedia 条目提到：用个人使命宣言（personal mission statement）去澄清方向，并持续回顾；“万事两次创造”——先在头脑中创造，再在现实中创造。

3. **Put first things first（要事第一）**
   - 要点：在“重要/紧急”之间做优先级管理。
   - Wikipedia 条目提到 Eisenhower matrix（重要性 × 紧急性）四象限，并强调很多人时间被象限 III/IV 吞噬；完成象限 I 后，应投入更多到“重要但不紧急”的象限 II（长期建设）。

4. **Think win–win（双赢思维）**
   - 要点：在人际与合作中寻求互利的方案与协议。
   - Wikipedia 条目将其描述为一种以品格为基础的人际协作方式，而非“讨好”或速成技巧。

5. **Seek first to understand, then to be understood（知彼解己）**
   - 要点：先用同理心倾听并理解对方，再表达自己。
   - Wikipedia 条目提到：这种倾听会促使对方回报式倾听，形成更有利的问题解决氛围；并用 ethos/pathos/logos（信誉/共情/逻辑）来说明说服的次序与重要性。

6. **Synergize（统合综效 / 协同增效）**
   - 要点：通过积极的团队合作，把个体优势组合起来，达到单靠个人无法实现的结果。

7. **Sharpen the saw（不断更新）**
   - 要点：在长期主义下持续“更新”自己，让前面六个习惯可持续。
   - Wikipedia 条目举例：身体更新（锻炼）、精神/心智更新（阅读等）、灵性更新（服务社会等）。
   - 条目还提到“upward spiral（向上螺旋）”模型：learn → commit → do，通过持续学习与实践实现成长与改进。

## 条目中的结构性分组

- Wikipedia 条目写到：作者将 **前 3 个习惯**视为达到“独立（independence）”的路径，**接下来 3 个**用于达到“互赖（interdependence）”，**第 7 个**用于维护与更新前面的习惯。

## 影响力（Wikipedia 条目提到的事实）

- Wikipedia 条目提到：2011 年《Time》将其列入“最具影响力的商业管理书籍”之一。
- 条目还提到：截至 2012 年作者去世时，该书销量超过 2000 万册。
